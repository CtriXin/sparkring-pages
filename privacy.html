隐私政策

SparkRing (思路集) · 最后更新: 2026 年 3 月 26 日

概述

SparkRing 是一款 AI 模型对比工具。我们高度重视您的隐私。本政策说明我们如何处理您的数据。

我们不收集的数据

  • 我们不收集您的姓名、邮箱或任何个人身份信息
  • 我们不追踪您的使用行为或浏览历史
  • 我们不使用广告追踪或第三方分析工具
  • 我们不访问您的相机、麦克风、通讯录或位置信息

本地存储的数据

以下数据仅存储在您的设备本地，不会上传到任何服务器：

  • 对话记录 — 您与 AI 模型的所有对话内容
  • API Key — 使用 AES-256-GCM 加密后存储在设备本地的 IndexedDB 中
  • 偏好设置 — 主题、模型选择等个人偏好

体验通道 (SparkRing)

为了提供免配置的体验功能，App 会生成一个随机的安装实例标识符 (install_id)，用于分配体验额度。该标识符：

  • 不与您的真实身份关联
  • 不会用于追踪或广告
  • 仅用于防止额度滥用
  • 卸载 App 或清除数据后会重新生成

第三方 API 服务

当您使用 AI 对话功能时，您的提问内容会发送到您所选择的第三方 AI 模型提供商（如 OpenAI、Google、Anthropic 等）。这些数据的处理受相应提供商的隐私政策约束。SparkRing 不会存储或中转这些对话数据。

数据删除

您可以随时在「设置」页面点击「清除所有敏感数据」来删除所有本地存储的 API Key。卸载 App 将清除所有本地数据。

儿童隐私

SparkRing 不面向 13 岁以下儿童。我们不会故意收集儿童的个人信息。

政策变更

如果本政策有重大变更，我们会在 App 内通知您。
