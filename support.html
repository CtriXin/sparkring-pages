常见问题

如何开始使用？

打开 App 后，点击「先试试看」→「立刻开始」，即可使用内置体验通道，无需配置任何 API Key。

什么是 API Key？一定要配置吗？

API Key 是访问 AI 模型服务的凭证。如果您只想体验基本功能，无需配置。如果您希望使用更多模型或更大额度，可以在「设置」中添加自己的 API Key。

我的数据安全吗？

所有对话记录和 API Key 均存储在您的设备本地。API Key 使用 AES-256-GCM 加密存储。我们不收集您的个人数据。

体验额度用完了怎么办？

体验通道提供有限的免费额度。额度用完后，您可以配置自己的 API Key 继续使用。支持 OpenAI、Google Gemini、Anthropic Claude、DeepSeek 等主流服务商。

支持哪些 AI 模型？

支持所有 OpenAI 兼容接口的模型，包括但不限于 GPT-4o、Gemini 2.5 Pro、Claude Sonnet/Haiku、DeepSeek、通义千问等。您也可以添加自定义 API 端点。

如何清除我的数据？

前往「设置」页面底部，点击「清除所有敏感数据」即可删除所有 API Key。卸载 App 将清除所有本地数据。
